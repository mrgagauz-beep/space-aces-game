# Space Aces Bot

Space Aces Bot — экспериментальный бот для игры в стиле DarkOrbit, использующий компьютерное зрение и эмуляцию ввода для автоматизации фарма мобов и сбора бонус‑коробок.

> ВАЖНО: бонускоробки **НЕ отображаются** на миникарте и детектируются **только на основном экране**; миникарта используется для навигации и поиска врагов.

## Установка

1. Установите Python 3.10+.
2. Клонируйте репозиторий и перейдите в папку проекта.
3. Установите зависимости:

```bash
pip install -r requirements.txt
```

## Запуск

1. Откройте игру Space Aces в оконном или полноэкранном режиме.
2. Убедитесь, что ROI в `profiles/default.json` соответствует расположению **основного экрана** и **миникарты**.
3. Запустите бота:

```bash
python run.py
```

Рекомендуемые платформы: Windows / Linux с поддержкой захвата экрана и эмуляции клавиатуры/мыши.

## Структура

- `run.py` — точка входа.
- `profiles/default.json` — профиль с ROI, HSV‑порогами и раскладкой клавиш.
- `assets/templates/` — спрайты/иконки/шаблоны (мобы, портал, ящик).
- `assets/screenshots/` — скриншоты для отладки и датасета.
- `logs/` — лог‑файлы работы бота.
- `src/vision/` — захват экрана, работа с миникартой и основным экраном, шаблоны.
- `src/controls/` — управление мышью и клавиатурой с лёгкой рандомизацией.
- `src/nav/` — навигация по миникарте, переходы между точками.
- `src/core/` — FSM (FARMING / FLEEING / SAFE и др.).
- `src/utils/` — логгер и вспомогательные утилиты.
- `src/config/` — загрузка/валидация профиля, доступ к ROI, порогам и клавишам.

## Важные оговорки

- Использование бота может нарушать правила игры — вы действуете на свой риск.
- Мобы распознаются по:
  - миникарте (цвет/форма меток);
  - основному экрану (шаблоны / CV; позже — нейросети).
- Бонус‑коробки распознаются **только** на основном экране по цвету/шаблонам.
- Навигация выполняется по миникарте, бой и сбор — по основному экрану.
