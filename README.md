# Space Aces Bot

Space Aces Bot — это бот для игры **Space Aces**, который автоматизирует рутину: сбор бонусов, фарм врагов и управление кораблём, чтобы вы могли меньше кликать и больше наслаждаться результатом.

## Возможности

- Захват экрана для анализа игрового процесса в реальном времени.
- Обнаружение врагов и бонус-коробок на экране.
- Автоматическое управление кораблём:
  - стрельба по врагам;
  - сбор ресурсов и бонус-коробок;
  - уклонение при опасности (низкое здоровье, скопление врагов и т.п.).

## Установка

1. Убедитесь, что установлен **Python 3.8+**.
2. Склонируйте репозиторий или скачайте исходный код.
3. Установите зависимости:

   ```bash
   pip install -r requirements.txt
   ```

## Запуск

1. Откройте игру **Space Aces** и выведите окно игры на экран (желательно без перекрытий).
2. Убедитесь, что выбран нужный профиль настроек (по умолчанию используется `profiles/default.json`).
3. Запустите бота из корня проекта:

   ```bash
   python run.py
   ```

Бот разрабатывается с прицелом на работу под **Windows** и **Linux**. Убедитесь, что у вас есть доступ к захвату экрана и всем необходимым правам для автоматизации ввода.

## Настройки

### Файл `profiles/default.json`

В этом файле описывается профиль настроек бота по умолчанию. Здесь можно настроить:

- регион экрана (ROI, Region of Interest) — область, в которой находится окно игры;
- горячие клавиши для управления кораблём (стрельба, способность, ускорение и т.п.);
- пороги здоровья и безопасности (когда начинать уклоняться или отступать);
- чувствительность и пороги детекции (насколько уверенно бот должен распознавать врагов/бонусы);
- скорость реакций и частоту обновления логики.

Вы можете создавать собственные профили в `profiles/`, копируя `default.json` и изменяя параметры под свой экран, раскладку и стиль игры.

### Инструмент калибровки ROI

В каталоге `tools` планируется скрипт `calibrate_roi.py`, который поможет:

- визуально выделить область игры на экране;
- сохранить координаты и размеры ROI в профиль (`profiles/default.json` или другой выбранный профиль);
- быстро перенастроить бота при смене разрешения экрана или расположения окна игры.

## Структура проекта

- `src/vision/` — модули компьютерного зрения: захват экрана, поиск врагов, бонус-коробок, анализ состояния интерфейса.
- `src/controls/` — управление кораблём: эмуляция клавиатуры/мыши, выполнение действий по решениям ядра.
- `src/core/` — основная логика бота: цикл обработки, принятие решений, состояние, поведение ИИ.
- `src/utils/` — вспомогательные функции и утилиты: логирование, тайминг, общие хелперы.
- `src/config/` — конфигурация и загрузка настроек, валидация профилей, общие параметры.
- `profiles/` — JSON-профили настроек бота (например, `default.json` для стандартной конфигурации).
- `tools/` — вспомогательные инструменты разработки и настройки, включая `calibrate_roi.py` для калибровки ROI и прочие утилиты.
- `run.py` — точка входа для запуска бота.
